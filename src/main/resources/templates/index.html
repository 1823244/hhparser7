Приложение - агрегатор вакансий с hh.ru

Позволяет вести статистику вакансий в разрезе "Проектов" и "Строк поиска"
"Проект" - это сущность, которая объединяет несколько "Строк поиска",
чтобы охватить все вакансии в каком-то направлении.
Например, проект Java может включать такие строки поиска:
java developer
java architect
java teamlead

Инструкция

Добавить проект на странице projects
Строка поиска должна соединять слова знаком плюс (+)
Пример: java+teamlead
Это не баг, так принимает данные api.hh.ru
